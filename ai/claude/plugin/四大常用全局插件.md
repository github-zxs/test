# （Claude Code 内置指令）

## 一、code-format 代码格式化

**作用**：一键格式化全项目 / 指定文件，统一缩进、换行、空格

### 常用指令

1. 格式化**整个项目**

plaintext

```
/format all
```

2. 格式化**当前打开文件**

plaintext

```
/format
```

3. 格式化指定目录 / 文件

plaintext

```
/format src/main/java
```

4. 仅格式化代码，不改动注释

plaintext

```
/format code-only
```

### 适配语言

Java、TS、Vue、Python、Go、JS 全支持，自动识别项目 ESLint/Prettier/Google Java 风格

---

## 二、git-helper Git 快捷助手

**作用**：不用手敲 git 命令，AI 一键完成版本操作

### 常用指令

1. 查看当前分支状态、修改文件

plaintext

```
/git status
```

2. 一键暂存 + 提交（自动写简洁提交信息）

plaintext

```
/git commit
```

3. 自定义提交文案提交

plaintext

```
/git commit 修复登录接口bug
```

4. 拉取远端最新代码

plaintext

```
/git pull
```

5. 推送本地分支到远端

plaintext

```
/git push
```

6. 查看提交日志、版本记录

plaintext

```
/git log
```

7. 自动解决简单代码冲突

plaintext

```
/git resolve conflict
```

8. 切换分支

plaintext

```
/git checkout 分支名
```

---

## 三、terminal-helper 终端命令助手

**作用**：自动生成、执行项目编译、打包、启动、运维命令

### 常用指令

1. 运行项目启动命令（自动识别框架）

plaintext

```
/run start
```

2. 项目打包编译

plaintext

```
/run build
```

3. 清空缓存 / 依赖

plaintext

```
/run clean
```

4. 执行测试命令

plaintext

```
/run test
```

5. 查端口占用、杀进程

plaintext

```
/run kill port 8080
```

6. Linux / 服务器运维常用命令生成

plaintext

```
/run 查看磁盘占用
/run 解压压缩包
/run 后台运行程序
```

7. 直接让 AI 帮你执行任意终端需求

plaintext

```
/run 帮我配置本地环境变量
```

---

## 四、search-files 全局文件检索

**作用**：极速搜索项目代码、注释、接口、关键字

### 常用指令

1. 全局搜索关键字

plaintext

```
/search 关键字
```

示例：

plaintext

```
/search UserLogin
/search 数据库连接
/search TODO
```

2. 只搜指定后缀文件

plaintext

```
/search 关键字 .java
/search 关键字 .vue
```

3. 只搜注释里内容

plaintext

```
/search note:待优化
```

4. 搜索接口地址、URL

plaintext

```
/search url:/api/user
```

5. 精准查找函数 / 方法

plaintext

```
/search function:getUserInfo
```

---

# 快捷使用总结

1. 格式化统一用：**/format**
2. Git 所有操作前缀：**/git**
3. 跑项目、敲命令前缀：**/run**
4. 找代码找内容前缀：**/search**

# 安装统一命令（全局生效）

plaintext

```
/plugin install code-format --scope user
/plugin install git-helper --scope user
/plugin install terminal-helper --scope user
/plugin install search-files --scope user
```
