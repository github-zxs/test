## ui获取token后的处理
  **后端登录**: const { data: loginToken, error } = await fetchLogin(loginDto);
  -  登录成功
	  1. loginByToken(loginToken: Api.Auth.LoginToken) 
		  - localStg.set('token', loginToken.access_token);  把token存储到 localStorage 其它访问的url 开始携带token信息
		  - getUserInfo() 获取用户信息( Object.assign(userInfo, info);)和权限信息  (userInfo.permissions = permissions);
		  - 跳转到home 路由
	  2. 登录认证失败 resetStore() 
		  - 重置 store相关信息
		  - 返回到当前路由
