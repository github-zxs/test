Ubuntu 24.04 中 Fcitx5 完整安装与配置方案

好，直接给你 **Ubuntu 24.04 专用、最稳的 Fcitx5 完整安装 + 配置方案**，一条一条复制就行。

---

## 1. 安装 Fcitx5 全套（拼音 + 云拼音 + 五笔 + 界面）

```bash
sudo apt update && sudo apt install -y fcitx5 fcitx5-configtool fcitx5-chinese-addons fcitx5-pinyin fcitx5-table-wubi fcitx5-module-cloudpinyin fcitx5-frontend-gtk3 fcitx5-frontend-qt5
```

## 2. 切换输入法框架为 fcitx5

```bash
im-config -n fcitx5
```

## 3. 环境变量（Ubuntu 24.04 必加）

```bash
echo 'GTK_IM_MODULE=fcitx5' >> ~/.pam_environment
echo 'QT_IM_MODULE=fcitx5' >> ~/.pam_environment
echo 'XMODIFIERS=@im=fcit5' >> ~/.pam_environment
echo 'INPUT_METHOD=fcitx5' >> ~/.pam_environment
```

## 4. 重启电脑（必须）

```bash
reboot
```

## 5. 开机后配置输入法

打开：

```bash
fcitx5-configtool
```

- 左下角 **+**
    
- 关掉 **Only Show Current Language**
    
- 搜索 **Pinyin** 或 **Wubi** 添加即可
    

## 6. 快捷键

- Ctrl + Space：中英文切换
    
- Super + Space：切换输入法
    

---

### 如果你用 GNOME（Ubuntu 默认）

托盘图标不显示：

```bash
sudo apt install gnome-shell-extension-kimpanel
```

然后打开 Extensions 开启 **Kimpanel**。
